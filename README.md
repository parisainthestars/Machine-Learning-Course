# Machine-Learning-Course

This repository contains my machine learning course assignments. Each homework focuses on a different classical or modern machine learning topic, including regression, classification, neural networks, dimensionality reduction, clustering, SVMs, and machine unlearning.

| HW | Topic | Main Methods / Models | More Details |
|---|---|---|---|
| HW1 | Regression, Logistic Classification, and Ensembles | Linear Regression, Ridge, Lasso, Logistic Regression, Naive Bayes, LDA, AdaBoost | [README](./HW1%20-%20Regression%2C%20Logistic%20Classification%2C%20and%20Ensembles/README.md) |
| HW2 | Neural Networks, CNNs, and Autoencoders | MLP, CNN, FashionMNIST, MNIST Autoencoder | [README](./HW2%20-%20Neural%20Networks%2C%20CNNs%2C%20and%20Autoencoders/README.md) |
| HW3 | PCA, SVM, and Clustering | PCA, Eigenfaces, SVM Kernels, K-means, t-SNE | [README](./HW3%20-%20PCA%2C%20SVM%2C%20and%20Clustering/README.md) |
| HW4 | Machine Unlearning and Backdoor Attacks | SISA, CIFAR-10, ResNet18, Membership Inference, Backdoor Poisoning | [README](./HW4%20-%20Machine%20Unlearning%20and%20Backdoor%20Attacks/README.md) |

## Covered Topics

- Regression and regularization
- Binary and multinomial classification
- Ensemble learning
- Neural networks from scratch
- Convolutional neural networks
- Autoencoders
- Principal Component Analysis
- Support Vector Machines
- K-means clustering and t-SNE visualization
- Machine unlearning and backdoor attacks

## Repository Structure

Each homework folder contains the related notebooks/code, figures, and a dedicated `README.md` with a short explanation of the implementation, results, and key takeaways.
